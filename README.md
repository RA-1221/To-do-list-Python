# Task Management System

A Python-based Task Management System built using Object-Oriented Programming and JSON file storage.

## Features

- Add tasks
- View tasks
- Search for tasks
- Edit task titles
- Edit task priorities
- Delete tasks
- Mark tasks as done
- Save tasks using JSON
- Load saved tasks when the program starts

## Project Progress

### Version 1.0
The initial version of the project was a basic Task Management System built with Python and Object-Oriented Programming.

### Version 2.0
The project was updated by adding JSON file storage, allowing tasks to be saved and loaded between program sessions.

## Technologies

- Python
- Object-Oriented Programming (OOP)
- JSON
- File Handling
